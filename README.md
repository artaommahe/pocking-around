## Development

### Packages

#### Add

package

```sh
cargo new --vcs none packages/p<XXX>-<name>
```

library

```sh
cargo new --vcs none --lib libs/<name>
```

#### Run

```sh
cargo run -p p<XXX>-<name>
```

or

```sh
cd packages/p<XXX>-<name>
cargo run
```
